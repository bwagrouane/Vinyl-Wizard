# Vinyl Wizard

A music reccomendation website that allows you to type in an album and get 9 reccomendations for an album, Built with HTML CSS Javascript and Awesomeplete, Cover Art Archive and Kanye REST API


## Features

You can type in an album that exists within AOTY's top 5000 rated albums and it will display the 9 albums titles, authors and album covers
Also my homepage generates a random kanye quote in the format of minecrafts splash text, just cause i thought itd be funny
